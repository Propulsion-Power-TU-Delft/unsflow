# Compressor Instability Prediction Tools
The folder is organized in different modules. Every module is independent from the others, except Sun, which needs data objects obtained by grid.

### Greitzer Model (1976)
##### Reference article: Surge and Rotating Stall in Axial Flow Compressors—Part I: Theoretical Compression System Model - Greitzer
 
Contains scripts to solve the Greitzer model ODEs and to perform a linearized stability analysis.

### Greitzer-Moore Model (1984-1985)
##### Reference article: A Theory of Post-Stall Transients in Axial Compression Systems: Part II—Application - Greitzer, Moore

Contains scripts to solve the Greitzer-Moore model ODEs.

### Spakovszky Model (2000)
##### Reference thesis: Applications of axial and radial compressor dynamic system modeling - Spakovszky, PhD thesis

Contains the classes and methods to perform an incompressible 2D linearized stability analysis.

### Sun Model (2013)
##### Reference article: A General Theory of Flow-Instability Inception in Turbomachinery - Sun et al.

Contains the classes and methods to perform a compressible 3D linearized stability analysis.
